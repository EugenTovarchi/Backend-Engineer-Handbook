# Backend Engineer Handbook

Инженерный справочник для подготовки к Middle+/Senior .NET Backend интервью.

Цель проекта — не собрать набор разрозненных ответов, а построить связанную базу знаний:

- простое объяснение темы;
- техническое описание;
- практические примеры из проектов;
- типичные ошибки;
- вопросы для собеседования;
- краткая шпаргалка.

## Основной стек

- C# / .NET / ASP.NET Core
- PostgreSQL
- RabbitMQ
- Redis
- Docker / Docker Compose
- Nginx
- EF Core / Dapper
- Quartz.NET
- S3 / MinIO
- Auth / JWT / Claims / Policies

## Рабочий процесс

1. Анализируем существующие Obsidian-заметки.
2. Выделяем готовые темы, дубли и пробелы.
3. Переносим материал в единую структуру книги.
4. Каждую главу приводим к единому формату.
5. Ведём статус проекта в `00_Project/PROJECT_STATUS.md`.
